Cyberra UI Kit

Компоненты для форм, редактора писем и отчётов. Сам input не получает верхний отступ: расстояние задаётся контейнером поля.

Размеры

XXS 19px XS 22px S 28px M 32px L 40px

Для кнопок из Figma это не универсальная шкала по всем вариантам: text buttons идут как `XXS=19`, `S=32`, `M=36`, `L=40`, а icon-only как `XXS=18`, `S=28`, `M=32`, `L=40`.

Кнопки

Синхронизировано с компонентом `Button` в Figma: text buttons `XXS=19`, `L=40`, `M=36`, `S=32`; icon-only `XXS=18`, `L=40`, `M=32`, `S=28`; радиус 4px, текст 13/16 Regular, `light` = `#EDF2F6`, `error` = `#FE495B`, `error light` = `#FFeded / #FE495B`, hover/pressed через opacity `0.85`, disabled — `0.4`. На клике у `ui-btn` теперь есть ripple-эффект.

JS API

window.UIKit.createButton({ size: 'm', variant: 'danger', text: 'Удалить' })
window.UIKit.createInput({ placeholder: 'Название' })
window.UIKit.createCheckbox({ label: 'Value', checked: true })
window.UIKit.createColorField({ id: 'bg', label: 'Цвет фона', value: '#101010' })
window.UIKit.createDropdown({
  value: 'm',
  items: [
    { value: 's', label: 'S' },
    { value: 'm', label: 'M' }
  ]
})
